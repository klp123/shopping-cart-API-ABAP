🛒 Shopping Cart API

RESTful API for the ABAP Shopping Cart POC — JWT-secured endpoints

Base URL: http://localhost:3000/api/v1
🔑
All endpoints except POST /auth/login require an Authorization: Bearer <token> header. Obtain the token from the login endpoint first.
Authentication
POST /auth/login Authenticate and receive a JWT No Auth ▶
Request Body
// Content-Type: application/json
{
  "email":    "user@example.com",  // required, valid email
  "password": "secure-password"    // required
}
Response — 200 OK
{
  "token": "eyJhbGciOiJIUzI1NiIsInR5..."
}
200 OK 400 Validation Error 401 Invalid Credentials
Products
GET /products Fetch product catalogue from SAP OData 🔒 Bearer ▶
Query Parameters
ParamTypeRequiredDescription
categorystringoptionalFilter by product category (e.g. Electronics)
minPricenumberoptionalMinimum price filter
maxPricenumberoptionalMaximum price filter
limitnumberoptionalMax records to return ($top) — UI defaults to 8
skipnumberoptionalRecords to skip ($skip) for pagination
Example Request
GET /api/v1/products?limit=8&skip=0&category=Electronics

Authorization: Bearer eyJhbGci...
Response — 200 OK
{
  "count": 42,
  "products": [
    {
      "id":       1,
      "name":     "Wireless Headphones",
      "price":    2999,
      "category": "Electronics",
      "stock":    15
    }
  ]
}
200 OK 401 Unauthorized
Cart
GET /cart/:sessionId Return cart with totals 🔒 Bearer ▶
Path Parameters
ParamTypeRequiredDescription
sessionIdstringrequiredUnique cart session identifier
Example Request
GET /api/v1/cart/sess-1714600000-abc123

Authorization: Bearer eyJhbGci...
Response — 200 OK
{
  "sessionId": "sess-1714600000-abc123",
  "items": [
    {
      "itemId":   "item-uuid",
      "productId": "prod-001",
      "name":      "Wireless Headphones",
      "price":     2999,
      "quantity":  2,
      "lineTotal": 5998
    }
  ],
  "subtotal":  5998,
  "discount":  0,
  "total":     5998,
  "coupon":    null
}
200 OK 401 Unauthorized
POST /cart/:sessionId/items Add a product to the cart 🔒 Bearer ▶
Request Body
// Content-Type: application/json
{
  "productId": "prod-001",   // required, non-empty string
  "name":      "Wireless Headphones", // required
  "price":     2999,          // required, positive number
  "quantity":  2              // required, int 1–10
}
Response — 201 Created
// Returns the full updated cart (same shape as GET /cart/:sessionId)
{
  "sessionId": "sess-...",
  "items":     [...],
  "subtotal":  5998,
  "discount":  0,
  "total":     5998
}
201 Created 400 Validation Error 401 Unauthorized
PUT /cart/:sessionId/items/:itemId Update quantity of a cart item 🔒 Bearer ▶
Request Body
// Content-Type: application/json
{
  "quantity": 3   // required, int 1–10
}
Path Parameters
ParamDescription
sessionIdCart session ID
itemIdCart item UUID (from GET cart response)
Response — 200 OK
// Returns the full updated cart
{
  "sessionId": "sess-...",
  "items":     [...],
  "subtotal":  8997,
  "discount":  0,
  "total":     8997
}
200 OK 400 Validation Error 401 Unauthorized 404 Item Not Found
DEL /cart/:sessionId/items/:itemId Remove a single item from the cart 🔒 Bearer ▶
Path Parameters
ParamDescription
sessionIdCart session ID
itemIdCart item UUID to remove
No request body required.
Response — 200 OK
// Returns the full updated cart (item removed)
{
  "sessionId": "sess-...",
  "items":     [...],
  "subtotal":  2999,
  "total":     2999
}
200 OK 401 Unauthorized 404 Item Not Found
POST /cart/:sessionId/coupon Apply a discount coupon code 🔒 Bearer ▶
Request Body
// Content-Type: application/json
{
  "code": "SAVE10"   // required, non-empty string
}
Response — 200 OK
// Returns cart with discount applied
{
  "sessionId": "sess-...",
  "items":     [...],
  "subtotal":  5998,
  "discount":  500,
  "total":     5498,
  "coupon":    "SAVE10"
}
200 OK 400 Invalid Coupon 401 Unauthorized

Orders
POST /orders Create an order from the current cart 🔒 Bearer ▶
Request Body
// Content-Type: application/json
{
  "sessionId": "sess-1714600000-abc123", // required
  "userId":    "user-1"                  // required
}
Response — 201 Created
{
  "orderId":   "order-uuid",
  "userId":    "user-1",
  "status":    "confirmed",
  "items":     [...],
  "subtotal":  5998,
  "discount":  0,
  "total":     5998,
  "createdAt": "2026-05-02T10:30:00.000Z"
}
201 Created 400 Empty Cart / Validation 401 Unauthorized

Error Response Format
All error responses follow a consistent structure ▶
Generic Error (401, 404, 500…)
{
  "error": "Invalid or expired token"
}
Validation Error (400)
{
  "error": "Validation failed",
  "details": [
    {
      "field":   "price",
      "message": "price must be a positive number"
    }
  ]
}
